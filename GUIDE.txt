APUNTES PARCIAL - DESARROLLO DE SOFTWARE  

A) CLONAR UN REPOSITORIO DE GITHUB: 
	1. Obtener la URL del Repositorio (Primero, necesitas la dirección para clonar.)
	2. Abrir tu Terminal/Línea de Comandos.
	3. Navega al directorio donde quieres que se guarde el repositorio clonado.  
	4. Ejecutar el Comando: "git clone [URL_Repositorio]"
	5. Entrar a la carpeta que fue clonada "cd nombre-del-repositorio"
B) COMUNICACIÓN CON OTRO USUARIO DESDE GITHUB: 
	1. Comunicación con otra persona por medio de 'ISSUES' en donde se pueden describir como una serie de problemas, para aclarar con mi compañero.  
C) CREACIÓN DE UN ENTORNO VIRTUAL: 
	1. Navegar a la carpeta del proyecto. 
	2. Verificar la versión de PYTHON: "python --version | py --version | python3 --version" (En Git Bash)
	3. Crear el entorno virtual "python -m venv venv"
	4. Activar el entorno virtual: "source venv/Scripts/activate" | "source venv/bin/activate"
D) SUBIR UN DIRECTORIO A GIT HUB:
	1. Inicializar GIT en la carpeta que se esta trabajando: "git init"
	2. Preparar (STAGE) y confirmar (COMMITS): "git add ."
	3. Confirmación de archivos: "git commit -m "INICIAL COMMIT""
	4. Especificar la rama principal: "git branch -M main"
	5. Conectar con el repositorio: "git remote add origin [LINK_REPOSITORIO]"
	6. Subir archivos: "git push -u origin main"
